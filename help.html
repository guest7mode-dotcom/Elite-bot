Need Help?

If you are confused about points, classes, cards or punishments, use these steps to get support.

1. Basic questions

  • Check the rules channel in Discord for regulations and examples.
  • Use commands like !help or !rules inside the server to see bot help.

2. Open a ticket

  • If something is bugged or unfair, open a support ticket in the Headmaster / Help channel.
  • Explain what happened, which command you used, and attach screenshots if possible.

3. Contact staff directly

  • Only ping staff for serious issues (exploits, harassment, account problems).
  • Do not spam DMs or channels; one clear report is enough.
